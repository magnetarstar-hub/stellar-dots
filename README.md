## 📁 Contents
- `hypr/` — Hyprland, hyprlock, hypridle, shaders, workflows
- `waybar/` — Status bar
- `rofi/` — App launcher
- `kitty/` — Terminal
- `nvim/` — Neovim
- `fish/` — Fish shell config
- `gtk-3.0/` & `gtk-4.0/` — GTK theming (Catppuccin Mocha)

## ⚙️ Installation
```bash
git clone https://github.com/magnetarstar/stellar-dots.git
cd stellar-dots
cp -r * ~/.config/
```
